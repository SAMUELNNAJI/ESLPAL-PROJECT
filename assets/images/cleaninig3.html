1 UPDATE the image section HTML

Replace your current image section with this:

html
<div class="image-section">

  <!-- Hidden radio buttons -->
  <input type="radio" name="robot-color" id="white" checked>
  <input type="radio" name="robot-color" id="black">
  <input type="radio" name="robot-color" id="silver">

  <!-- Images -->
  <div class="robot-images">
    <img src="images/robot-white.png" class="img white-img" alt="White Robot">
    <img src="images/robot-black.png" class="img black-img" alt="Black Robot">
    <img src="images/robot-silver.png" class="img silver-img" alt="Silver Robot">
  </div>

  <!-- Color Switcher -->
  <div class="color-badges">
    <label for="white" class="badge white"></label>
    <label for="black" class="badge black"></label>
    <label for="silver" class="badge silver"></label>
  </div>

</div>

Make sure you have 3 robot images:

  • robot-white.png

  • robot-black.png

  • robot-silver.png

2 ADD this CSS (image switching logic)

Add this below your image-section styles:

css
/* Hide radio buttons */
.image-section input {
  display: none;
}

/* Image container */
.robot-images {
  position: relative;
  width: 100%;
  max-width: 400px;
}

.robot-images img {
  width: 100%;
  position: absolute;
  opacity: 0;
  transition: opacity 0.4s ease, transform 0.4s ease;
  filter: drop-shadow(0 20px 30px rgba(0,0,0,0.35));
}

/* Default image */
#white:checked ~ .robot-images .white-img,
#black:checked ~ .robot-images .black-img,
#silver:checked ~ .robot-images .silver-img {
  opacity: 1;
  position: relative;
  transform: scale(1.02);
}

/* Color badges */
.color-badges {
  margin-top: 25px;
  display: flex;
  gap: 14px;
}

.badge {
  width: 22px;
  height: 22px;
  border-radius: 50%;
  cursor: pointer;
  border: 2px solid white;
  transition: transform 0.2s, box-shadow 0.2s;
}

.badge:hover {
  transform: scale(1.2);
  box-shadow: 0 0 10px rgba(255,255,255,0.8);
}

.badge.white { background: #fff; }
.badge.black { background: #000; }
.badge.silver { background: #cfcfcf; }
